## 常用命令行

* git init

* git clone -b develop https://username:password@remote

* git checkout -b xxx (创建并切到本地xxx分支)   

* git checkout xxx (切换分支)

* git merge bxx (当前所在分支合并bxx分支的修改)

* git branch -d xxx  (删除本地xxx分支, 当前所在分支是xxx以外的分支)
